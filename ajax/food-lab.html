Food Lab

Project Info

This is a single page project aimed to create a visually striking and mobile-responsive landing page for a fictional food company, "Food Lab". The objective was to implement a given design that would showcase the company's services in an intuitive and easy-to-navigate layout.

By focusing solely on front-end development, the project utilized various tools such as Javascript, React, HTML, and CSS to create a seamless user experience across all devices.
